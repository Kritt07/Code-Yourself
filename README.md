# Code Yourself
Code Yourself — это пошаговая головоломка с элементами платформера, где игрок не управляет персонажем напрямую, а пишет набор команд (псевдокод), которые затем выполняются автоматически.

Игрок управляет квадратным персонажем, задача которого — пройти уровень, полный препятствий (пилы, движущиеся объекты, ловушки), используя заранее прописанный алгоритм действий.

## Ключевая идея игры:

Игрок должен не реагировать, а предугадывать и программировать решение.

### Каждый уровень — это:

- задача на логику
- задача на тайминг
- задача на оптимизацию команд
